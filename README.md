<h1 align="center">🫀 Heart Disease Prediction Using Machine Learning</h1>

<p align="center">
A machine learning classification project to predict heart disease based on patient medical data.
</p>

<hr>

<h2>📖 Project Overview</h2>

<p>
Heart disease is one of the leading causes of death worldwide. This project aims to build a machine learning model that predicts whether a person has heart disease using medical attributes. Multiple ML algorithms are trained and evaluated to identify the best-performing model.
</p>

<hr>

<h2>📊 Dataset Information</h2>

<ul>
<li><b>Dataset:</b> Heart_Diseases_Prediction.csv</li>
<li><b>Problem Type:</b> Binary Classification</li>
<li><b>Target Variable:</b> 0 (No Disease), 1 (Disease)</li>
</ul>

<h3>Features</h3>

<ul>
<li>Age</li>
<li>Sex</li>
<li>Chest Pain Type</li>
<li>Resting Blood Pressure</li>
<li>Cholesterol</li>
<li>Fasting Blood Sugar</li>
<li>Resting ECG</li>
<li>Maximum Heart Rate</li>
<li>Exercise Induced Angina</li>
<li>ST Depression</li>
<li>Slope</li>
<li>Number of Major Vessels</li>
<li>Thalassemia</li>
</ul>
